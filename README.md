# MapBiomas Fire Brazil
Developed by [Amazon Environmental Research Institute - IPAM](https://ipam.org.br/pt/)

## About
**This is the official repository for the MapBiomas Fire project in Brazil.**

This repository contains the scripts, tools, and methodologies used for mapping burned areas in Brazil, currently supporting the MapBiomas Fire Collection 3 project. 

For detailed information about the classification and methodology applied in Brazil, please refer to the MapBiomas Fire [Appendix of the Algorithm Theoretical Basis Document (ATBD)](https://brasil.mapbiomas.org/metodo-mapbiomas-fogo/).

## Repository Objectives
* Share the methods and code used for mapping burned areas in Brazil.
* Serve as the central hub for the MapBiomas Fire network, facilitating collaboration and adaptation by international partners.

## Repository Structure Overview

To keep the repository organized according to different initiatives and data collections, the structure is divided into the following main directories:

* **[`network/`](./network/)**: Concentrates the effort of sharing our methodology with other countries that have joined the MapBiomas network (MapBiomas Fire Network). It contains generalized algorithms and structures.
* **[`access_tools/`](./access_tools/)**: Contains scripts, interfaces, and tools designed to facilitate data access and interaction for users.
* **[`mapbiomas_fire_collections/`](./mapbiomas_fire_collections/)**: Holds the legacy and active processing structures specific to Brazilian fire collections (e.g., `collection_04`, `collection_05`).
* **[`mapbiomas_fire_monitor/`](./mapbiomas_fire_monitor/)**: Contains scripts and structures used for the continuous monitoring of fire events (e.g., `version_01`).
