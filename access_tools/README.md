# MapBiomas Fire Access Tools
Developed by [Amazon Environmental Research Institute - IPAM](https://ipam.org.br/pt/)

## About

This directory, **`access_tools/`**, is dedicated to scripts, APIs, interfaces, and automated tools that facilitate access to MapBiomas Fire data and analytical resources.

As the MapBiomas network grows and the volume of processed data expands across collections (`mapbiomas_fire_collections`) and continuous monitoring solutions (`mapbiomas_fire_monitor`), having dedicated and streamlined access mechanisms is essential for users, researchers, and policymakers.
