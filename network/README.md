# MapBiomas Fire Network
Developed by [Amazon Environmental Research Institute - IPAM](https://ipam.org.br/pt/)

## About
**This is the root directory for the MapBiomas Fire Network project.** 

This initiative represents the expansion of the MapBiomas Fire network to other countries in South America. The content in this directory focuses on the effort of sharing our methodology and facilitating collaboration internationally.

The methodology is designed to be replicable across different regions of the continent, using Deep Neural Network (DNN) algorithms for model training and classification.

## Versions and Directory Structure

To maintain the evolution of the project while ensuring backward compatibility with legacy operations, the network algorithms are organized into versions:

### [version_01/](./version_01/) (Legacy)
This directory contains the original scripts and methods utilized for the first collection of data across the network (Collection 1). It is maintained for historical reference and compatibility purposes.

### [version_02/](./version_02/) (Active)
This represents the current and active standard for the network (Collection 2 onwards). This version brings enhanced models, automated parameterizations, expanded scopes, and multi-country operational flows.
- **Quick Start via Colab**: The `mapbiomas_fire_classification_v1.ipynb` notebook located inside `version_02/` serves as the primary entry point to easily test and adapt the most recent classification scripts without local setup.
